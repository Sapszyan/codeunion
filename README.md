# `Currencies Project`

### Configuration Setup
In root directory create a configuration file named `.env` based on the provided `.env.example`.

## Migration
```
yarn migrate
```

## Currencies update
```
yarn update-currency
```

## Usage

```
yarn start
```

> [ Open [http://localhost:3000](http://localhost:3000) to test it in browser ]

## Dev-Mode start

```
yarn dev
```
> [ Open [http://localhost:3000](http://localhost:3000) to test it in browser ]
