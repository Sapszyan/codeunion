# `Currencies Project`

## Installation

`Make sure you created .env`

`Copy from .env.example to .env`

## Migration
```
yarn migrate
```

## Currencies update
```
yarn update-currency
```

## Usage

```
yarn start
```

> [ Open [http://localhost:3000](http://localhost:3000) to test it in browser ]

## Dev-Mode start

```
yarn dev
```
> [ Open [http://localhost:3000](http://localhost:3000) to test it in browser ]
